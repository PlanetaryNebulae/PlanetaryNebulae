Project 1: Website

The first project I have to share is... this website! Originally, this was a website for one of my game ideas, and it used material from the game concept document. Now, it's been slightly altered to work for my portfolio.

This was designed to be a dynamic website that can be viewed on multiple screen sizes, and it uses a three-column layout. There is a slight zoom-in animation on all images, and a special animation I made for the logo.

The original website also had a table, drop-down selections on the navigation, and a web form that used check boxes, radio buttons, text area, and drop-down selections that could be submitted or reset.
